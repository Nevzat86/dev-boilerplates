# Dev Boilerplates Bundle

Two production-ready boilerplates for shipping fast.

## Included

### 1. Chrome Extension MV3 Boilerplate
React + TypeScript + Vite + Tailwind CSS. Includes popup, side panel, options page, service worker, content scripts, OAuth2, BYOK API pattern, and Gumroad license validation.

→ [chrome-extension-mv3/README.md](chrome-extension-mv3/README.md)

### 2. WordPress Plugin + React Admin Boilerplate
PHP + React + @wordpress/scripts + @wordpress/components. Includes admin dashboard, settings framework, REST API, custom post types, and full security patterns.

→ [wp-plugin-react-admin/README.md](wp-plugin-react-admin/README.md)

## Quick Start

```bash
# Chrome Extension
cd chrome-extension-mv3
pnpm install
pnpm dev

# WordPress Plugin
cd wp-plugin-react-admin
pnpm install
pnpm start
```
